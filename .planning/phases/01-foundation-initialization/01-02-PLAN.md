  **main.tex template:**
  Create a compilable abnTeX2 LaTeX document that uses biblatex+biber. Use `{{PLACEHOLDER}}` syntax (custom template engine -- NOT LaTeX braces).

  ```latex
  % Main thesis file - Generated by Get Thesis Done
  % Compile with: latexmk -pdf -bibtex-cond1 main.tex
  %
  \documentclass[
    12pt,
    openright,
    oneside,
    a4paper,
    chapter=TITLE,
    section=TITLE,
    english,         % Secondary language
    {{LANGUAGE_BABEL}}  % Primary language (last = active): e.g., brazil, english, spanish
  ]{abntex2}

  % --- Packages ---
  \usepackage[utf8]{inputenc}
  \usepackage[T1]{fontenc}
  \usepackage{lmodern}
  \usepackage{graphicx}
  \usepackage{microtype}
  \usepackage{hyperref}
  \usepackage{bookmark}

  % --- Bibliography (biblatex + biber) ---
  % Decision: biber+biblatex over bibtex+natbib for UTF-8 support
  \usepackage[
    backend=biber,
    style=abnt,        % ABNT citation style (requires biblatex-abnt package)
    % style=apa,       % Uncomment for APA style
    noslsn,
    uniquename=init,
  ]{biblatex}
  \addbibresource{references.bib}

  % --- Thesis metadata ---
  \titulo{{{THESIS_TITLE}}}
  \autor{{{AUTHOR_NAME}}}
  \orientador{{{ADVISOR_NAME}}}
  % \coorientador{{{CO_ADVISOR_NAME}}}  % Uncomment if applicable
  \instituicao{{{INSTITUTION}}}
  \local{{{CITY}}}
  \data{{{YEAR}}}
  \tipotrabalho{{{WORK_TYPE}}}  % e.g., Dissertacao (Mestrado), Tese (Doutorado), TCC
  \preambulo{{{PREAMBLE}}}  % e.g., "Dissertacao apresentada ao Programa de..."

  % --- Document ---
  \begin{document}
  \selectlanguage{{{LANGUAGE_BABEL}}}
  \frenchspacing

  % --- Pre-textual elements ---
  \pretextual
  \imprimircapa
  \imprimirfolhaderosto*

  % Cataloging sheet, approval sheet, dedication, acknowledgments, epigraph
  % (uncomment and fill as needed)
  % \include{pre-textual/ficha-catalografica}
  % \include{pre-textual/folha-aprovacao}
  % \include{pre-textual/dedicatoria}
  % \include{pre-textual/agradecimentos}
  % \include{pre-textual/epigrafe}

  % Abstract(s)
  \begin{resumo}
    {{ABSTRACT_PT}}

    \textbf{Palavras-chave}: {{KEYWORDS_PT}}
  \end{resumo}

  \begin{resumo}[Abstract]
    \begin{otherlanguage*}{english}
      {{ABSTRACT_EN}}

      \textbf{Keywords}: {{KEYWORDS_EN}}
    \end{otherlanguage*}
  \end{resumo}

  % Lists
  \pdfbookmark[0]{\listfigurename}{lof}
  \listoffigures*
  \cleardoublepage
  \pdfbookmark[0]{\listtablename}{lot}
  \listoftables*
  \cleardoublepage

  % Table of contents
  \pdfbookmark[0]{\contentsname}{toc}
  \tableofcontents*
  \cleardoublepage

  % --- Textual elements ---
  \textual
  {{CHAPTER_INCLUDES}}
  % Example: \include{chapters/01-introduction}

  % --- Post-textual elements ---
  \postextual
  \printbibliography[title={Refer\^encias}]

  % Appendices and annexes (uncomment as needed)
  % \begin{apendicesenv}
  %   \include{appendices/appendix-a}
  % \end{apendicesenv}
  % \begin{anexosenv}
  %   \include{annexes/annex-a}
  % \end{anexosenv}

  \end{document}
  ```

  IMPORTANT NOTES:
  - The `style=abnt` requires `biblatex-abnt` package (part of TeX Live). This is the correct way to use ABNT with biblatex+biber.
  - The `{{PLACEHOLDER}}` syntax uses double curly braces which do NOT conflict with LaTeX's single curly braces.
  - All `\titulo{}`, `\autor{}` etc. are abnTeX2 commands, not standard LaTeX.
  - The template must compile with placeholders replaced by simple text. Leave structure clean.
