**1. parseFiguresCatalog(content)** -- Parse the Figures table from FIGURES.md content.
- Find the `## Figures` section up to `## Tables` (or end of file)
- Split into lines, skip header row (`| ID |`), separator row (`|---|`), empty rows, and comment lines
- For each data row: split by `|`, trim cells, filter empty strings
- Require at least 6 cells with a non-empty first cell
- Return array of `{ id, caption, chapter, type, sourceFile, status }` objects
- If no `## Figures` section found, return empty array

**2. appendFigureEntry(content, entry)** -- Append a figure entry row to FIGURES.md content.
- Build new row string: `| ${entry.id} | ${entry.caption} | ${entry.chapter} | ${entry.type} | ${entry.sourceFile} | ${entry.status} |`
- Check if the content has an empty placeholder row (regex: `/(\| *\| *\| *\| *\| *\| *\|)/`)
  - If found: replace the FIRST empty row with the new row
  - If not found: find `## Tables` section and insert the new row on a new line BEFORE it (with a preceding newline)
  - If no `## Tables` section: append the new row at the end of content
- Return the modified content string

**3. extractFigureRefs(texContent)** -- Extract all `\ref{fig:*}`, `\autoref{fig:*}`, and `\Autoref{fig:*}` from .tex content.
- Use three regex patterns: `/\\ref\{fig:([^}]+)\}/g`, `/\\autoref\{fig:([^}]+)\}/g`, `/\\Autoref\{fig:([^}]+)\}/g`
- Collect all matches into a Set (deduplicates)
- Return the Set of bare IDs (without `fig:` prefix)

**4. extractFigureLabels(texContent)** -- Extract all `\label{fig:*}` from .tex content.
- Use regex: `/\\label\{fig:([^}]+)\}/g`
- Collect into a Set, return it

**5. normalizeToKebabCase(input)** -- Normalize a figure ID to kebab-case.
- Convert camelCase to kebab-case: insert hyphen before uppercase letters, then lowercase
- Replace underscores with hyphens
- Replace multiple consecutive hyphens with a single hyphen
- Trim leading/trailing hyphens
- Lowercase the entire result
- Examples: `systemArch` -> `system-arch`, `system_arch` -> `system-arch`, `System-Arch` -> `system-arch`

**6. cmdRegisterFigure(cwd, args, raw)** -- FIG-01/FIG-04: Register a figure in FIGURES.md.
- Parse arguments:
  - `--id <id>` (required) -- the figure ID, will be normalized to kebab-case
  - `--type <excalidraw|tikz|static>` (required) -- must be one of these three values
  - `--chapter <N>` (required) -- chapter number
  - `--caption <text>` (required) -- figure caption text
- Validate all required args; error with usage string if any missing
- Validate type is one of: excalidraw, tikz, static; error if not
- Normalize ID via normalizeToKebabCase
- Determine source file path based on type:
  - excalidraw: `figures/${id}.excalidraw`
  - tikz: `figures/${id}.tikz`
  - static: `figures/${id}.png` (default extension; user can change later)
- Determine initial status: `planned`
- Read FIGURES.md from `.planning/FIGURES.md`; error if not found ("FIGURES.md not found. Run /gtd:new-thesis first.")
- Parse existing catalog via parseFiguresCatalog to check for duplicate ID
- If ID already exists in catalog, error: "Figure with ID '{id}' already exists in FIGURES.md"
- Call appendFigureEntry to add the new row
- Write updated content back to FIGURES.md
- Output JSON: `{ id, caption, chapter, type, sourceFile, status: 'planned', catalogPath: '.planning/FIGURES.md' }`
- Raw output: "Registered figure '{id}' (type: {type}) in FIGURES.md for chapter {chapter}"

**7. cmdValidateFigs(cwd, raw)** -- FIG-05: Cross-reference validation of figure refs against catalog.
- Read `.planning/FIGURES.md`, error if not found
- Parse catalog via parseFiguresCatalog to get registered figure IDs
- Scan .tex files in TWO locations (same pattern as cmdValidateRefs):

- For each .tex file: extract figure refs via extractFigureRefs, also extract figure labels via extractFigureLabels
- Aggregate:
  - `allReferencedIds`: Set union of all `\ref{fig:*}` and `\autoref{fig:*}` across all .tex files
  - `allLabeledIds`: Set union of all `\label{fig:*}` across all .tex files
  - `catalogIds`: Set of IDs from parsed FIGURES.md
- Calculate:
  - `missingFromCatalog`: IDs referenced in .tex files but not in FIGURES.md catalog (orphaned references)
  - `unreferencedInCatalog`: IDs in FIGURES.md catalog but never referenced in any .tex file (informational warning)
  - `labeledButNotInCatalog`: IDs that have `\label{fig:*}` in .tex but no catalog entry
- Per-chapter breakdown: for each scanned .tex file, report ref count and any missing IDs
- Output JSON: `{ total_catalog_entries, total_referenced_ids, missing_from_catalog: [], unreferenced_in_catalog: [], labeled_not_in_catalog: [], per_chapter: [{ file, source: 'approved'|'draft', ref_count, label_count, missing_ids: [] }] }`
- Raw output: formatted report (same style as cmdValidateRefs):
  - "Figure Validation Report" header
  - Catalog entries count, referenced IDs count
  - Per-chapter breakdown with OK/ISSUES status
  - Missing from catalog list (if any), or "All figure references found in FIGURES.md."
  - Unreferenced catalog entries list (if any, as informational note)

**Part A: Add exportExcalidraw helper and implement preProcessFigures in gtd-tools.js**

**1. exportExcalidraw(srcPath, exportPath)** -- Export a single .excalidraw file to PDF.
Place this function right before preProcessFigures.
- Ensure export directory exists: `fs.mkdirSync(path.dirname(exportPath), { recursive: true })`
- Try excalirender first (preferred -- self-contained binary):
  - Check availability: `execSync('which excalirender', { stdio: 'pipe' })`
  - Run export: `execSync('excalirender "' + srcPath + '" -o "' + exportPath + '"', { stdio: 'pipe', timeout: 30000 })`
  - Return `{ success: true, method: 'excalirender' }`
- If excalirender fails, try fallback (excalidraw-to-svg + rsvg-convert):
  - Check rsvg-convert availability: `execSync('which rsvg-convert', { stdio: 'pipe' })`
  - If rsvg-convert not found, return `{ success: false, method: 'none', error: 'No Excalidraw export tool found. Install one of:\n  excalirender: curl -fsSL https://raw.githubusercontent.com/JonRC/excalirender/main/install.sh | sh\n  rsvg-convert: sudo apt-get install librsvg2-bin' }`
  - Compute SVG intermediate path: `exportPath.replace(/\.pdf$/, '.svg')`
  - Run: `execSync('npx --yes excalidraw-to-svg "' + srcPath + '" "' + path.dirname(svgPath) + '"', { stdio: 'pipe', timeout: 60000 })`
  - Run: `execSync('rsvg-convert -f pdf -o "' + exportPath + '" "' + svgPath + '"', { stdio: 'pipe', timeout: 15000 })`
  - Return `{ success: true, method: 'excalidraw-to-svg + rsvg-convert' }`
- If both fail, return `{ success: false, method: 'fallback', error: err.message }`
